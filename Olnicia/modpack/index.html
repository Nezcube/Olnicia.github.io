Tutoriel pour installer le modpack

  • Télécharger Tlauncher
  • Faire Win + R puis taper %appdata%
  • Cliquer sur le dossier .minecraft
  • Aller dans le dossier versions
  • Glisser le dossier Olnicia présent dans l'archive
  • Ouvrir Tlauncher puis selectioner la version "Release Olnicia"
  • Selectioner "Multiplayer" et Rejoindre le serveur déjà dans le menu
